# 유튜브 스팸 댓글 필터

[English README](./README.md)

유튜브 영상 페이지에서 스팸으로 보이는 댓글을 숨겨주는 Chrome 확장 프로그램입니다. 영상 업로더가 작성한 댓글과 좋아요가 많은 댓글은 필터링 대상에서 제외합니다.

## 주요 기능

- 유튜브 영상 페이지 댓글을 직접 필터링합니다.
- 스크롤하면서 새로 로드되는 댓글도 감지합니다.
- 확장 프로그램 팝업에서 필터를 켜고 끌 수 있습니다.
- 검사한 댓글 수와 숨긴 댓글 수를 표시합니다.
- 숨긴 댓글을 흐리게 다시 표시하는 디버그 보기를 제공합니다.
- 댓글을 보면서 `Spam` 또는 `Not spam` 피드백을 남길 수 있습니다.
- 사용자의 피드백을 바탕으로 로컬에서 필터링을 개선합니다.
- 팝업에서 로컬 학습 통계를 확인하고 학습 데이터를 초기화할 수 있습니다.
- 영상 업로더가 작성한 댓글은 필터링하지 않습니다.
- 좋아요가 많은 댓글은 필터링하지 않습니다. 기본 기준은 좋아요 100개입니다.
- `ㅋㅋㅋㅋ`, `ㅎㅎㅎㅎ`, `!!!!!` 같은 단순 반복 표현만으로는 스팸 처리하지 않습니다.

## 필터링 방식

이 확장 프로그램은 로컬 규칙 기반 휴리스틱을 사용합니다. 댓글 텍스트를 외부 서버로 전송하지 않습니다.

현재 사용하는 주요 스팸 신호는 다음과 같습니다.

- 의심스러운 링크
- 텔레그램, 카카오톡, WhatsApp, Discord, DM, 프로필 확인 유도
- 투자, 코인, 수익, 부업, forex 관련 홍보
- giveaway, airdrop, 쿠폰, 당첨, 이벤트 홍보
- 클릭, 가입, 참여, claim 같은 행동 유도 문구
- 홍보성 단어 반복
- 여러 개의 링크
- 과도한 해시태그 또는 멘션
- 매우 짧은 홍보성 문구

필터링은 점수 기반입니다. 따라서 일반적인 링크 하나나 단순 반복 감탄 표현만으로는 댓글을 숨기지 않도록 설계했습니다.

## 로컬 학습

이 확장 프로그램은 외부 프로그램이나 서버 없이 사용자의 피드백을 통해 로컬에서 학습할 수 있습니다.

- 댓글 피드백은 Chrome 확장 프로그램 로컬 저장소에 저장됩니다.
- 댓글 원문은 저장하지 않습니다. 댓글 해시, 감지된 필터 신호, 사용자 피드백 라벨만 저장합니다.
- 같은 댓글 텍스트에 대한 직접 피드백은 이후 판정에 우선 반영됩니다.
- 반복된 피드백은 `link`, `contact`, `money-promo` 같은 신호의 가중치를 조정합니다.
- 학습 데이터는 사용자의 브라우저 안에만 남고 다른 사용자와 공유되지 않습니다.
- 팝업에서 모든 학습 데이터를 초기화할 수 있습니다.

## 로컬 실행

의존성을 설치합니다.

```powershell
npm install
```

검증 명령을 실행합니다.

```powershell
npm run check
npm test
npm run build
```

## Chrome에서 로드하기

1. `chrome://extensions`를 엽니다.
2. Developer mode를 켭니다.
3. Load unpacked를 클릭합니다.
4. 생성된 `dist` 폴더를 선택합니다.
5. 유튜브 영상 페이지를 열고 댓글 영역까지 스크롤합니다.
6. 확장 프로그램 팝업에서 검사한 댓글 수와 숨긴 댓글 수를 확인합니다.

소스 코드를 수정한 뒤에는 다시 빌드하고 확장 프로그램을 새로고침해야 합니다.

```powershell
npm run build
```

## 한계

- 이 확장 프로그램은 사용자의 브라우저 화면에서만 댓글을 숨깁니다.
- 유튜브 댓글을 삭제, 신고, 수정하지 않습니다.
- 머신러닝 모델이나 외부 검열 API를 사용하지 않습니다.
- 유튜브 DOM 구조가 바뀌면 selector 업데이트가 필요할 수 있습니다.
- 규칙 기반 필터링이므로 오탐 또는 미탐이 발생할 수 있습니다.
